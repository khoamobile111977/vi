Xử Lý & Đếm Ký Tự Trong Xâu

Viết chương trình nhập xâu ký tự và thống kê số lượng chữ số, chữ cái trong xâu bằng Python.

Yêu Cầu Đề Bài

Viết chương trình yêu cầu nhập từ bàn phím một xâu ký tự, sau đó thông báo:

01 Yêu cầu 1: Tổng số các ký tự là chữ số (0–9) trong xâu.
02 Yêu cầu 2: Tổng số các ký tự là chữ cái tiếng Anh (a–z, A–Z) trong xâu.
→ Viết hàm riêng cho mỗi yêu cầu trên.

Bước 2

Hàm tongchuso() — Giải Thích Từng Dòng

tongchuso.py
def tongchuso(xau):
    tong = 0
    for i in xau:
        if i >= '0' and i <= '9':
            tong = tong + 1
    return tong
def tongchuso(xau):

Khai báo một hàm tên là tongchuso. Hàm nhận vào một tham số là xau — đây là xâu ký tự mà ta muốn đếm.

tong = 0

Tạo biến tong và gán bằng 0. Đây là bộ đếm — ban đầu chưa đếm được gì nên bắt đầu từ 0.

for i in xau:

Vòng lặp for duyệt qua từng ký tự trong xâu. Mỗi lần lặp, biến i sẽ là một ký tự trong xâu.

if i >= '0' and i <= '9':

Kiểm tra xem ký tự i có phải chữ số không. Nếu i nằm trong khoảng từ '0' đến '9' thì đúng là chữ số.

tong = tong + 1

Nếu điều kiện trên đúng thì tăng bộ đếm lên 1. Ví dụ: tong đang là 2 → thành 3.

return tong

Trả về kết quả đếm được. Sau khi duyệt hết xâu, tong chính là tổng số chữ số.

Bước 3

Hàm tongchucai() — Giải Thích Từng Dòng

tongchucai.py
def tongchucai(xau):
    tong = 0
    for i in xau:
        if (i >= 'a' and i <= 'z') \
          or (i >= 'A' and i <= 'Z'):
            tong = tong + 1
    return tong
def tongchucai(xau):

Khai báo hàm tên tongchucai. Tương tự hàm trước, nhận vào xau là xâu cần đếm chữ cái.

tong = 0

Khởi tạo bộ đếm bằng 0. Mỗi lần tìm thấy chữ cái, biến này sẽ được tăng thêm 1.

for i in xau:

Duyệt lần lượt từng ký tự trong xâu. Biến i lần lượt nhận giá trị từng ký tự một.

if (i >= 'a' and i <= 'z') or (i >= 'A' and i <= 'Z'):

Điều kiện gồm 2 vế nối bằng or: vế trái kiểm tra chữ thường (a–z), vế phải kiểm tra chữ hoa (A–Z). Chỉ cần một trong hai đúng là tính.

tong = tong + 1

Ký tự hiện tại là chữ cái → tăng bộ đếm lên 1.

return tong

Trả về tổng số chữ cái đã đếm được sau khi duyệt hết toàn bộ xâu.

Code

Chương Trình Hoàn Chỉnh

baitap.py
# ─── Hàm đếm chữ số ───────────────
def tongchuso(xau):
    tong = 0
    for i in xau:
        if i >= '0' and i <= '9':
            tong = tong + 1
    return tong

# ─── Hàm đếm chữ cái ──────────────
def tongchucai(xau):
    tong = 0
    for i in xau:
        if (i >= 'a' and i <= 'z') \
          or (i >= 'A' and i <= 'Z'):
            tong = tong + 1
    return tong

# ─── Chương trình chính ───────────
xau = input("Nhap xau: ")
print("Tong chu so: ", tongchuso(xau))
print("Tong chu cai: ", tongchucai(xau))
